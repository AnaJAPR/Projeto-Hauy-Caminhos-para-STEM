# Guia de Instalação — Python, VS Code e Git

Este guia vai te ajudar a preparar seu computador para programar em Python.

Não se preocupe se algo não funcionar de primeira, faz parte do aprendizado!

## Instalando Git Bash

Siga o passo a passo do [site oficial](https://git-scm.com/install) de acordo com o seu sistema operacional.
Se for windows, por exemplo, clique na aba **Windows** e no primeiro link que aparece. Vai instalar um executável.
Após o download, clique no arquivo e clique em "next" até começar a instalar o Git Bash.
Para saber se deu certo, procure pelo app na barra de pesquisa do seu notebook ou computador.

## Instalando VS Code

Acesse https://code.visualstudio.com/, clique em **Download** e escolha o seu sistema operacional. Instale normalmente aceitando as opções padrão.

### Extensões no VS Code

1. Abra o VS Code

2. Clique no ícone de extensões que fica na lateral esquerda

3. Pesquise "Python"

4. Instale a extensão oficial da Microsoft

Para instalar qualquer extensão necessária em algum trabalho, os passos são os mesmos, mas ao invés de pesquisar "Python", pesquise o que precisar.

## Instalando Python

Para usar a linguagem de programação Python, precisamos instalar o **interpretador**, que é o programa que entende os comandos que escrevemos.

### Passo a passo (Windows / Linux / Mac)

1. Acesse o site oficial do Python: https://www.python.org/downloads/

2. Clique no botão **Download Python** (a versão mais recente).

3. Durante a instalação:
   - **Marque a opção “Add Python to PATH”**
   - Clique em **Install Now**

Essa opção é muito importante! Sem ela, o Python pode não funcionar no terminal.

### Testando se deu certo

1. Abra o **Git Bash** ou o **Terminal**
2. Digite:

```bash
python --version
```
3. Se aparecer algo como **Python 3.13.9**, deu certo! Se não, verifique novamente os passos acima.
